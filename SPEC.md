# ClawSharp (爪锐) 技术规格说明书 (SPEC) 索引

为了提高文档的可维护性和可读性，我们将 SPEC 拆分为以下模块：

- [**01 - 架构与项目概览**](docs/specs/01-architecture.md)
  - 包含核心愿景、技术栈和开发路线图。
- [**02 - 推理核心与流程**](docs/specs/02-core-engine.md)
  - 详细说明 ReAct 循环逻辑和记忆管理机制。
- [**03 - 安全规范与沙箱**](docs/specs/03-security.md)
  - 涵盖权限模型、审批流和审计日志。
- [**04 - NativeAOT 适配要求**](docs/specs/04-native-aot.md)
  - 针对 AOT 环境下的开发限制和优化策略。

---

## 如何贡献
在修改核心代码前，请确保已阅读并遵循对应模块的 SPEC。如需更新 SPEC，请提交 Pull Request 并关联相关 Issue。
